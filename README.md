ESCENA INICIAL
En esta escena se puede comenzar a jugar con la A y trastear con el jugador con la Z y se a dejado la posibilidad de disparar con las teclas: 
		B para disparar desde la camara(proyectil tipo 1)
		V para disparar desde la camara(proyectil tipo 2)
ESCENA DE JUEGO
Igual que en la escena inicial todo el control se realiza con la tecla .
EXTRAS
La cámara est bloqueada por defecto para desbloquearla pulsar B.
Para acceder a las escenas respectivas a las practicas puede hacerse con las teclas de menos y más (no son parte del proyecto final).
